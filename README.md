# CGI

Django aplikacija.

## Tehnologije

- Django 4.2.27
- PostgreSQL
- Cloudflare R2 (S3-compatible storage)
- SendGrid (email)
- CKEditor (rich text editor)
- WhiteNoise (static files)

## Funkcionalnosti

- ✅ Topics sistem sa rich text editorom
- ✅ User accounts
- ✅ Contact form sa SendGrid
- ✅ Email archiving
- ✅ Responsive design (mobile-first)
- ✅ Custom Django admin

## Struktura projekta

- `topics/` - Aplikacija za teme
- `accounts/` - User accounts
- `core/` - Home, contact, email management
- `templates/` - HTML template-i
- `static/` - CSS, JS, images

## Napomene

- Favicon i logo treba zameniti sa stvarnim fajlovima
- Kontakt informacije (telefon, adresa) treba ažurirati u template-ima
- Social media linkovi treba ažurirati sa stvarnim URL-ovima
